# SQL---DATASCRIPT
Datascript for databasen: Fiskekonkurranse
Denne ble brukt til eksamensoppgave i database på Kristiania Høyskole høsten 2025

Her finner du:
- Enkle SELECT-spørringer
- Hvordan man kombinerer data med JOIN
- Forskjellen mellom INNER JOIN, LEFT JOIN og RIGHT JOIN
- Hvordan Henrik og Gjedde viser disse forskjellene i praksis
- Hvordan man bruker GROUP BY, COUNT, SUM og AVG
- Hvordan HAVING filtrerer grupper
- Hvordan RIGHT JOIN fungerer
- Hvordan et VIEW kan gjøre komplekse spørringer enklere

Alt dette viser hvordan SQL brukes til både strukturert analyse og enkel rapportering.
